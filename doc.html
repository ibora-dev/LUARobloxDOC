1.1 – Les types de données

Lua utilise plusieurs types de données :

  • string : texte entre guillemets
  • number : nombres (entiers ou décimaux)
  • boolean : true ou false
  • nil : valeur vide
  • table : liste ou groupe d’éléments
  • function : bloc de code réutilisable
local pseudo = "Ibora"
local age = 17
local estConnecté = true
local rien = nil
local joueur = {nom = "Ibora", niveau = 5}
local function direBonjour()
  print("Salut !")
end

1.2 – Les variables

Une variable est un nom auquel on associe une valeur :

local score = 100
print(score)

Tu peux changer sa valeur :

score = score + 50

1.3 – Les conditions (if, else)

Les conditions servent à faire des choix dans le script :

local vie = 10

if vie > 0 then
  print("Le joueur est en vie")
else
  print("Le joueur est mort")
end

Avec plusieurs cas :

if vie > 50 then
  print("Plein de vie")
elseif vie > 0 then
  print("Blessé")
else
  print("Mort")
end
