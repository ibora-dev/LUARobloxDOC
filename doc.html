Partie 1 : Les bases du langage Lua

Chapitre 1.1 – Les types de données en Lua

En Lua, chaque donnée a un type. C'est ce qui permet au langage de savoir comment manipuler la valeur.

1. Texte (string)

Une string est une chaîne de caractères, c'est-à-dire du texte. Elle est toujours entre guillemets : "" ou ''.

Ici, la variable pseudo contient le texte Ibora.

2. Nombre (number)

Ce sont des valeurs numériques. Il n'y a pas de guillemets autour.

3. Booléen (boolean)

Un boolean représente le vrai ou le faux : true ou false.

4. Nil (vide)

nil signifie que la variable ne contient rien (valeur vide).

5. Table (liste ou groupe)

Les tables permettent de regrouper plusieurs valeurs.
