Chapitre 1.3 – Les conditions (if, else)

Les conditions te permettent de faire des choix dans ton script. C’est comme poser une question, et agir différemment selon la réponse.

La base : if

Ici, on demande : si la vie est plus grande que zéro, alors on affiche un message. Si la condition est vraie, le code entre if et end s’exécute.

Ajouter un sinon : else

Si la vie est plus grande que zéro, on affiche que le joueur est en vie. Sinon (si la vie est zéro ou moins), on affiche que le joueur est mort.

Plusieurs cas : elseif

Tu peux tester plusieurs conditions dans l’ordre. La première vraie condition est celle qui s’exécute.
